# موقع تعريفي شخصي - Personal Portfolio

موقع تعريفي شخصي احترافي مبني بـ PHP مع لوحة تحكم محمية لإدارة المشاريع والأعمال.

## المميزات

- ✨ تصميم عصري ومتجاوب
- 🔐 لوحة تحكم محمية بنظام مصادقة آمن
- 📁 رفع وإدارة المشاريع بسهولة
- 🖼️ رفع الصور للمشاريع
- 🎨 تصميم جميل باستخدام CSS مخصص
- 📱 متجاوب مع جميع الشاشات
- 🔒 حماية من هجمات SQL Injection و XSS
- 📊 قاعدة بيانات MySQL منظمة

## المتطلبات

- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- خادم Apache أو Nginx
- إضافة PDO PHP

## التثبيت والإعداد

### الخطوة 1: نسخ الملفات

```bash
# نسخ المشروع إلى مجلد الخادم
git clone <repository-url>
cd portfolio
```

### الخطوة 2: إنشاء قاعدة البيانات

1. افتح phpMyAdmin أو أي أداة إدارة قاعدة بيانات
2. قم باستيراد ملف `database.sql`
3. أو قم بتنفيذ الأوامر التالية في MySQL:

```sql
-- إنشاء قاعدة البيانات
CREATE DATABASE portfolio_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- استخدام قاعدة البيانات
USE portfolio_db;

-- نسخ محتوى ملف database.sql هنا
```

### الخطوة 3: تكوين الاتصال بقاعدة البيانات

افتح ملف `config/config.php` وقم بتحديث إعدادات قاعدة البيانات:

```php
define('DB_HOST', 'localhost');     // عنوان الخادم
define('DB_USER', 'root');          // اسم المستخدم
define('DB_PASS', '');              // كلمة المرور
define('DB_NAME', 'portfolio_db');  // اسم قاعدة البيانات
```

### الخطوة 4: إعداد صلاحيات المجلدات

```bash
# إعطاء صلاحيات الكتابة لمجلد uploads
chmod 755 uploads/
```

### الخطوة 5: الوصول للموقع

- **الصفحة الرئيسية:** `http://localhost/pro/index.php`
- **لوحة التحكم:** `http://localhost/pro/admin/login.php`

## معلومات تسجيل الدخول الافتراضية

```

كلمة المرور: admin123
```

**⚠️ مهم جداً:** يُنصح بشدة بتغيير كلمة المرور بعد أول تسجيل دخول!

## الاستخدام

### إضافة مشروع جديد

1. سجل الدخول إلى لوحة التحكم
2. اضغط على "إضافة مشروع جديد"
3. املأ معلومات المشروع:
   - عنوان المشروع (مطلوب)
   - الوصف
   - رابط المشروع
   - رابط GitHub
   - الفئة
   - التقنيات المستخدمة (افصل بفواصل)
   - صورة المشروع
   - ترتيب العرض
   - مشروع مميز (اختياري)
4. اضغط على "حفظ المشروع"

### تعديل إعدادات الموقع

1. سجل الدخول إلى لوحة التحكم
2. اضغط على "إعدادات الموقع"
3. قم بتحديث:
   - عنوان الموقع
   - وصف الموقع
   - معلومات "عني"
   - صورة الملف الشخصي
   - معلومات الاتصال
   - روابط وسائل التواصل الاجتماعي
4. احفظ التغييرات

## الأمان

الموقع يتضمن العديد من ميزات الأمان:

- ✅ حماية من SQL Injection باستخدام Prepared Statements
- ✅ حماية من XSS باستخدام htmlspecialchars
- ✅ التحقق من نوع وحجم الملفات المرفوعة
- ✅ التحقق من أن الملفات المرفوعة هي صور حقيقية
- ✅ تشفير كلمات المرور باستخدام password_hash
- ✅ نظام جلسات آمن
- ✅ حماية صفحات لوحة التحكم

### تحسينات الأمان الموصى بها

1. **تغيير كلمة المرور الافتراضية:**
   ```sql
   -- تشغيل في MySQL
   UPDATE users SET password = '$2y$10$NewHashedPasswordHere' WHERE username = 'admin';
   ```

2. **استخدام HTTPS:**
   - احصل على شهادة SSL
   - قم بتحديث `SITE_URL` في `config/config.php`
   - غير `session.cookie_secure` إلى 1

3. **تأمين مجلد uploads:**
   - أضف ملف `.htaccess` في مجلد uploads:
   ```apache
   # منع تنفيذ PHP في مجلد uploads
   php_flag engine off
   ```

## التخصيص

### تغيير الألوان

افتح `assets/css/style.css` أو `assets/css/admin.css` وعدل المتغيرات في `:root`:

```css
:root {
    --primary-color: #2563eb;     /* اللون الأساسي */
    --secondary-color: #1e40af;   /* اللون الثانوي */
    --success-color: #10b981;     /* لون النجاح */
    --danger-color: #ef4444;      /* لون الخطر */
    /* ... */
}
```

### إضافة حقول جديدة

1. عدل قاعدة البيانات
2. عدل النماذج في لوحة التحكم
3. عدل عرض البيانات في الصفحة الرئيسية

## استكشاف الأخطاء

### خطأ في الاتصال بقاعدة البيانات

- تحقق من صحة بيانات الاتصال في `config/config.php`
- تأكد من أن خادم MySQL يعمل
- تأكد من وجود قاعدة البيانات

### لا يمكن رفع الصور

- تحقق من صلاحيات مجلد `uploads/`
- تحقق من إعدادات PHP:
  ```ini
  upload_max_filesize = 10M
  post_max_size = 10M
  ```

### مشاكل في عرض الصفحات

- تأكد من تفعيل mod_rewrite في Apache
- تحقق من مسارات الملفات
